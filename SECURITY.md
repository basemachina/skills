# Security Policy

## Reporting a Vulnerability

セキュリティ上の脆弱性を発見した場合は、GitHub の [private vulnerability reporting](https://github.com/basemachina/skills/security/advisories/new) からご報告ください。GitHub Issue や Pull Request での公開報告は控えていただきますようお願いします。

報告いただいた内容は確認後、修正版のリリースおよび GitHub Security Advisory による公表を行います。
