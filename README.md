# basemachina/skills

BaseMachina 用の Agent Skill を配布するパブリックリポジトリ。

- ライセンス: Apache-2.0
- 対象エージェント: Claude Code / Codex CLI / Cursor / Copilot CLI / Gemini CLI 等

## 収録 skill

| skill | 概要 |
| --- | --- |
| [`bm-code-management`](skills/bm-code-management/) | `defineAction` を編集し `bm sync` で差分をプレビューする skill |

```bash
gh skill install basemachina/skills@bm-code-management
```
